Planowane aplikacje domenowe Django — dodawaj do `INSTALLED_APPS` wraz z implementacją.

- `accounts` — użytkownicy, role
- `tenants` — organizacje / multi-tenant
- `training` — pokoje, ścieżki, scenariusze
- `documents` — upload, chunkowanie
- `rag` — embeddingi, pgvector
- `ai` — wywołania LLM, prompty
- `assessments` — oceny, próby
- `analytics` — logi badawcze
- `adminops` — narzędzia superadmina

Obecnie zarejestrowane: `apps.core`.
